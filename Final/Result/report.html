Final Project

Motivation

Since Google released the first explorer wearable device in Feb 2013, more and more people has jumped into this area to develope their ideas about wearable computing devices. Moreover this device could provide a virtual screen which would be floating on the air and combining with the real scene to users. So it inspires me to use the techniques I learned from this course to simulate this amazing feature.

Algorithm

Left image shows how I image the path of light coming from the mini projector and the how the light will change the path on the special surface, combiner. The surface, combiner, has two special bsaf, specular-reflection and specular-transmission. So the eye will receive a light combining two light sources, ambient-light and projector-light.

System

- Light Source -

For the light source, I use the default projector light source as my base class. But because this class considers all the light is coming from a point instead of coming from a series of parallel light which is what I want. So I modify the direction of incoming light.

- Material -

For the material, I use the concept of light-splitter to implement this material. This material will have two specular propertities in its bsdf, specular reflection and specular transmittion. I also give this material a parameter to control the ratio between reflection and transmittion.

- Integrator -

I use the directlighting as my surface integrator. But because the default implementation about finding the intensity of light from sampling a point in bsdf always return zero. It makes the light coming from the splitter always become zero. So I design a special surface integrator for my system.
